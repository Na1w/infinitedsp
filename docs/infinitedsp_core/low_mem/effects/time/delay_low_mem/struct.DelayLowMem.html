Expand description

A memory-efficient digital delay effect using 16-bit integer storage and 2x downsampling.

Saves 75% memory compared to standard Delay by storing samples as i16 and running the delay line at half the sample rate. Uses SIMD-accelerated processing and Cubic (Hermite) interpolation to restore high-end.
