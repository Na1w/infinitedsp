Expand description

The core trait for all audio processors.

Implementors must handle processing a block of audio samples. The generic parameter C defines the channel configuration (Mono/Stereo).

Required Methods§

Source

fn process(&mut self, buffer: &mut [f32], sample_index: u64)

Processes a block of audio samples.

§Arguments
  • buffer - The audio buffer to process (in-place).
  • sample_index - The global sample index of the start of the block.

Resets the internal state of the processor.
