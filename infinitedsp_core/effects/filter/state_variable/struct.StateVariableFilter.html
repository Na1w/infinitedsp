Expand description

A State Variable Filter (SVF).

A stable and versatile filter that provides simultaneous low-pass, high-pass, band-pass and notch outputs. This implementation uses the TPT (Topology Preserving Transform) / ZDF (Zero Delay Feedback) method for excellent stability and response across the frequency range.
