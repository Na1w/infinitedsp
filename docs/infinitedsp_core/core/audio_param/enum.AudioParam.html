Enum AudioParam 

Source 
pub enum AudioParam {
    Static(f32),
    Dynamic(Box<dyn FrameProcessor<Mono> + Send>),
    Linked(Parameter),
}
Expand description

A parameter that can be static, dynamic (controlled by another processor), or linked to a thread-safe Parameter.

Variants§

§

Static(f32)

A constant value.

§

Dynamic(Box<dyn FrameProcessor<Mono> + Send>)

A value generated by another processor (e.g., LFO, Envelope). Parameters can only be Mono signals.

§

Linked(Parameter)

A value linked to a shared Parameter (e.g., UI control).

pub fn get_constant(&self) -> Option<f32>

Returns the constant value if the parameter is Static or Linked. Returns None if the parameter is Dynamic.

This allows processors to optimize for the common case where parameters are constant for the duration of a block.
